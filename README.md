# NewsCMS
基于SpringBoot的新闻管理发布系统，新闻后台管理系统。

## 项目介绍
本项目采用前后端分离，接口开发模式进行开发。前台使用[WeAdmin](https://github.com/WangEn/WeAdmin)来进行对接。

项目规模较小，主要用于划水期末jsp作业，如果能帮助到你，不妨给一个star支持一下。
